# study progress app

## 使用技術

- React
- TypeScript
- Vite

## 概要

勉強の進捗を管理するアプリケーションです。

## 工夫した点

- UIをシンプル化
- グラスモーフィズムの実装
- レスポンシブ対応

## 起動方法  

### 前提環境
  - Node.js 24.x
  - npm 11.x

### インストール

  リポジトリをクローンします。  
  > git clone `https://github.com/haz1109-ship-it/haz.github.io`

  プロジェクトフォルダへ移動します。  
  > cd study-progress-app

  必要なパッケージをインストールします。  
  > npm install  
  
### アプリの起動

  開発サーバーを起動します。  
  > npm run dev

  起動後、ブラウザで以下のURLを開きます。  
  > http://localhost:5173
